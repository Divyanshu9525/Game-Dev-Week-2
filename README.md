# Game Dev Week 2 Assignment

# Alien Character Rig and MultiState Character Animation 

#Description
    This project contains an alien character created in Blender fully rigged.
    Also contains a working Multistate Character Animation with Idle, Walk, Run, and Jump states in Unity with model and animations imported from mixamo, with one Animation Event trigger "Footsteps".
    Explored procedural animation techniques, but I was not able to complete a fully working procedural animation system within the project timeframe.
